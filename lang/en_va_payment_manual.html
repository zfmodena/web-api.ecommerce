How to Pay Through ATM Network

BCA/PRIMA ATM NETWORK

  1. Insert your debit card into ATM.
  2. Choose prefered language.
  3. Insert your card PIN.
  4. Choose "Other Transaction".
  5. Choose "Transfer".
  6. Choose "Others Bank Account".
  7. Insert Permata Bank code (013) and press "Continue".
  8. Insert payment amount with exact number as billed Rp#amount#.
    Transaction will fail if different amount inserted and you must repeat the process from beginning.
  9. Insert Payment Code #payment_code# and press "Continue".
  10. Confirmation screen will be shown contains:
    Permata Bank account number
    Payment recipient MODENA-#orderno#
    Payment amount
    If all displayed informations are correct, then press "Continue".
  11. Transaction is completed.
    You will receive transaction receipt.

MANDIRI/ATM BERSAMA NETWORK

  1. Insert your debit card into ATM.
  2. Choose prefered language.
  3. Insert your card PIN.
  4. Choose "Other Transaction".
  5. Choose "Transfer".
  6. Choose "Others ATM Bersama/Link Bank Account".
  7. Insert Bank Permata code (013) followed by Payment Code #payment_code#, then press "Continue".
  8. Insert payment amount with exact number as billed Rp#amount#.
    Transaction will fail if different amount inserted and you must repeat the process from beginning.
  9. You may skip entry for reference number and then press "Continue".
  10. Confirmation screen will be shown contains:
    Permata Bank account number
    Payment recipient MODENA-#orderno#
    Payment amount
    If all displayed information are correct, then press "Continue".
  11. Transaction is completed.
    You will receive transaction receipt.

PERMATA ATM

  1. Insert your debit card into ATM.
  2. Choose prefered language.
  3. Insert your card PIN.
  4. Choose "Other Transaction".
  5. Choose "Payment".
  6. Choose "Others".
  7. Choose "Virtual Account Payment".
  8. Insert Bank Permata code (013) followed by Payment Code #payment_code#, then press "Continue".
  9. Confirmation screen will be shown contains:
    Permata Bank account number
    Payment recipient MODENA-#orderno#
    Payment amount
    If all displayed information are correct, then press "Continue".
  10. Choose your registered account for payment.
  11. Transaction is completed.
    You will receive transaction receipt.
*. You might pay this bill using internet banking facility of your debit card issuer bank, only if available.
